# ToDoアプリ
タスクを登録し、<br>
確認、完了したタスクか未完了なタスクかを一目で確認することができます

## 機能一覧

- ユーザー登録、ログイン機能

- タスク追加機能

- カテゴリ追加機能

- タスクの完了確認機能

- 検索機能、カテゴリ検索機能

- ソート機能
